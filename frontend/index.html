Model Architecture
How ToxPredict Works
🕸️
Graph Convolutional Network
3 × GCNConv layers (hidden=128) on the molecular graph. Atoms are nodes (34-dim features: type, hybridisation, aromaticity). Bonds are edges. Global mean pooling → 2-layer MLP head → 12 logits.
⚗️
Molecular Featurisation
RDKit parses SMILES → atom/bond feature tensors via PyTorch Geometric. Each atom encodes type (one-hot 19-dim), degree, hybridisation, aromaticity, formal charge, Hs, ring membership.
📊
Tox21 Dataset
7,831 compounds × 12 toxicity endpoints (NR-AR, NR-ER, SR-ARE …). NIH/EPA, MIT license. Scaffold split (80/10/10). Masked BCE loss handles ~20% missing labels. Mean val AUROC = 0.831.
